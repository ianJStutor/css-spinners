50 CSS Animations

CSS-only animations direct from my imagination.
Only a single, empty HTML element. No SVGs. No images. No JavaScript.
Some styling uses experimental CSS rules that may not work in every browser.

Why?

At the beginning it was to see if I could. But after the first few it became something else: a tinkering playground for discovering obscure CSS rules and finding out what happens if I animate them. Above are the visual results, but I've retained some fascinating knowledge about how exactly CSS operates and, well, how it sometimes just doesn't work as expected.
